# 🎵 SPOTIFY-FULL-STACK 🎧  
A Full-Stack Music Streaming App built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js), mimicking core features of Spotify including song uploads, album management, and real-time playback.

---

## 🚀 Features

- 🎼 **Upload Songs** with metadata and images
- 💿 **Create & Manage Albums**
- 📂 **Store Media Files** with Cloudinary
- 🔊 **Music Player** with audio control
- 🗃️ **List, Play, and Delete** songs
- ⚙️ **RESTful APIs** with Express
- 🌐 **Frontend** in React + Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Tailwind CSS
- Axios
- React Context API (for global player state)

### Backend:
- Node.js
- Express.js
- MongoDB + Mongoose
- Cloudinary (image/audio upload)

---

## ⚙️ Setup Instructions

### 🔐 Prerequisites:
- Node.js & npm
- MongoDB Atlas or local MongoDB
- Cloudinary account (for media upload)

### 🔄 Clone the Repo

```bash
git clone https://github.com/koushik369mondal/SPOTIFY-FULL-STACK.git
cd SPOTIFY-FULL-STACK
```

### 🔧 Backend Setup

```bash
cd server
npm install
```
Start the backend:
```bash
npm run dev
```

### 🌐 Frontend Setup

```bash
cd client
npm install
npm start
```
